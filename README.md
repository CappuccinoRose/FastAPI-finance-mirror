# Finance Mirror Backend

一个为现代个人财务管理设计的后端服务，基于 FastAPI 构建，提供高效、安全的 API 接口。本项目从 MySQL 迁移至 PostgreSQL，并优化用于 Vercel 云平台的 Serverless 部署。

## 📋 功能特性

- 🏦 **总账管理**: 完整的会计科目体系，支持凭证与分录的创建、查询与管理。
- 💰 **应收应付**: 管理客户信息、销售发票，以及供应商信息和采购账单。
- 👥 **员工费用**: 支持员工账户创建及费用报告的提交与审批流程。
- 📊 **财务报表**: 提供资产负债表、利润表等核心财务报表的数据接口。
- 🔐 **认证授权**: 基于 JWT (JSON Web Tokens) 的安全用户认证与权限控制系统。
- ⚙️ **业务逻辑**: 统一的单据过账处理与自动化的会计期间结转功能。

## 🛠️ 技术栈

- **框架**: FastAPI (Python 3.13)
- **数据库**: PostgreSQL (推荐使用 Supabase)
- **ORM**: SQLAlchemy 2.0 (核心为异步)
- **数据库迁移**: Alembic
- **数据验证**: Pydantic V2
- **认证**: JWT (JSON Web Tokens)
- **部署**: Vercel (Serverless Functions)
